# ANSWER_FORMAT.md

## Формат ответов по проекту приложения

Все ответы по приложению Cozy Foodie должны быть только в этой структуре:

## 1. Что сделала

Кратко: какие действия выполнены, какая версия, что изменилось, что уже залито или подготовлено.

## 2. Идеи и предложения, что сделать

Кратко: 1-3 следующих логичных шага. Без большого списка и без лишней теории.

## 3. Вопросы ко мне, если есть

Если вопросов нет, писать: «Вопросов нет».

## 4. От меня

Пользователь отвечает:
- «окей» — подтверждение следующего шага;
- свои предложения / правки, если есть.

## Правило

Если пользователь пишет «окей», значит можно выполнять следующий предложенный шаг без повторного уточнения.
