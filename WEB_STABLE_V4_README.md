# Cozy Foodie web stable v4 recipes

Объединено с web stable v3 showcase.

Что добавлено:
- рецепты адаптированы под тип блюда;
- разные инструкции для пасты, супов, омлетов, курицы, рыбы, мяса, сладкого, салатов и т.д.;
- сохранены граммовки, огонь, время и юмор;
- шутки добавлены аккуратно, без перегруза;
- версия ассетов: webstable4.

Что проверить:
- открыть 10–15 разных рецептов;
- убедиться, что шаги соответствуют блюду;
- проверить сладкое, супы, пасту, омлеты;
- проверить список покупок.
